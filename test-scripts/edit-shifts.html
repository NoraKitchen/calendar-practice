Add Minimum Cumulative Skill Level

Minimum Cumulative Skill Level

What would you like the skill level of all employees on this shift to add up to?

Sometimes naming a minimum number of staff isn't enough. Two new hires cannot perform the same as two veterans.

Use this field to help you make sure you have enough talent on hand for a busy shift.

To use this functionality, make sure you assign skill levels to each of your employees.

Example
Employee Skill Level
Dasher 2
Dancer 1
Prancer 1
Vixen 4
Rudolph 10

If you set your minimum skill level to 7 and try to schedule inexperienced employees Dasher, Dancer, and Prancer, you will recieve a warning the minimum skill level is not met!

But if you schedule Dasher, Dancer, and Vixen, the minimum skill level has been met.

And if you schedule vetran Rudolph, he can get the job done all on his own!

Want to make sure at least one person on shift has a certain/minimum skill level? Check out the Special Requirements section below!
